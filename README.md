# how to test

First open [index.js](index.js) and set the `jwt` variable.

## 0. Testing metadata folder

Set the `src` variable to `"metadata"` and run:

```
npm install
node index
```

## 1. Testing image folder

Set the `src` variable to `"ipfs"` and run:

```
npm install
node index
```
